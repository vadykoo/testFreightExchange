## About “Біржа вантажів”

Це технічне завдання передбачає створення простого сервісу для додавання та
відображення вантажів.

## Технологічний стек

 ● Laravel
 ● Vue.js
 ● Bootstrap

## Як запустити?

composer install

npm install

php artisan migrate --seed
